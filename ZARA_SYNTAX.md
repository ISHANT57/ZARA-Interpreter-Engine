# ZARA Language Syntax

## Variables
Variables are declared using the `set` keyword.

Example:
set x = 10
set name = "ZARA"

---

## Arithmetic Operations
ZARA supports basic arithmetic operations: +, -, *, /

Example:
set x = 10
set y = 3
set result = x + y * 2

---

## Strings
Strings are written inside double quotes.

Example:
set message = "Hello World"

---

## Output
Use `show` to display values or text.

Example:
show result
show "Hello from ZARA"
